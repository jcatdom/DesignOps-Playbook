# Naming convention

## Naming is hard

### Colours

{% hint style="success" %}
Use **alert** for colour or component variation
{% endhint %}

{% hint style="danger" %}
Don't use **warning** as name for colour or component variation
{% endhint %}

### T-shirt sizes

{% hint style="success" %}
Use **S** **M** **L** **XL** **XXL** **XXXL** or **XL** **2XL** **3XL** for size and space
{% endhint %}

{% hint style="danger" %}
Don't use 2 letters: **SM** **MD** **LG** **XL**
{% endhint %}
